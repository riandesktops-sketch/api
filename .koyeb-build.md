# Zodiac AI Backend - Koyeb Deployment

## Build Configuration for Koyeb

### Build Command
```bash
go build -o app ./cmd/all-in-one
```

### Run Command
```bash
./app
```

### Environment Variables (REQUIRED)
```
MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/zodiac_ai
MONGODB_DATABASE=zodiac_ai
JWT_SECRET=your-super-secret-jwt-key-change-this
GEMINI_API_KEY=your-gemini-api-key
PORT=8080
```

### Port
```
8080
```

## Troubleshooting Build Errors

### Error: "cannot find package"
**Solution:** Pastikan semua dependencies sudah ada di `go.mod`
```bash
# Di local, run:
go mod tidy
git add go.mod go.sum
git commit -m "Update dependencies"
git push
```

### Error: "no Go files in /workspace"
**Solution:** Pastikan build command benar
```
Build command: go build -o app ./cmd/all-in-one
```

### Error: "cannot load package"
**Solution:** Check module name di `go.mod` harus `zodiac-ai-backend`
```go
module zodiac-ai-backend

go 1.21
```

## Quick Deploy Steps

1. **Push to GitHub**
   ```bash
   git add .
   git commit -m "Ready for Koyeb"
   git push origin main
   ```

2. **Koyeb Settings**
   - Builder: Buildpack (Go)
   - Build command: `go build -o app ./cmd/all-in-one`
   - Run command: `./app`
   - Port: 8080

3. **Set Environment Variables** (di Koyeb dashboard)

4. **Deploy!**
